### 3. Color Palette (Complex Design)

**Primary Colors:**
- Deep Blue: `#1a237e` (Headers, important text)
- Teal: `#00695c` (Accents, highlights)
- Coral: `#ff6f61` (Call-to-action, key metrics)

**Secondary Colors:**
- Light Gray: `#f5f5f5` (Backgrounds)
- Medium Gray: `#9e9e9e` (Supporting text)
- White: `#ffffff` (Text on dark backgrounds)

**Gradient Combinations:**
1. Header gradient: `#1a237e` вҶ’ `#283593`
2. Background gradient: `#f5f5f5` вҶ’ `#ffffff`
3. Accent gradient: `#00695c` вҶ’ `#00897b`

### 4. Typography System

**Font Hierarchy:**
```
H1 (Slide Titles):
- Font: Montserrat Bold / Roboto Bold
- Size: 44pt
- Color: #1a237e
- Letter spacing: -0.5pt

H2 (Section Headers):
- Font: Montserrat SemiBold
- Size: 32pt
- Color: #00695c
- Letter spacing: 0pt

H3 (Subsections):
- Font: Montserrat Medium
- Size: 24pt
- Color: #424242

Body Text:
- Font: Open Sans Regular / Roboto Regular
- Size: 18pt
- Color: #616161
- Line height: 1.5

Captions/Footer:
- Font: Open Sans Light
- Size: 12pt
- Color: #9e9e9e
```

### 5. Advanced Graphic Elements

#### Icons & Illustrations
- **Style:** Line icons with 2px stroke
- **Color:** Teal (#00695c) with 20% opacity background circle
- **Size:** 64x64px for main icons, 32x32px for supporting icons
- **Placement:** Left-aligned with 40px margin

#### Data Visualization
**Charts:**
- Bar charts: Gradient fills (primary color вҶ’ lighter shade)
- Pie charts: Distinct colors from palette with 3D subtle depth
- Line graphs: 3px stroke, gradient area fill below line
- Add data labels with white background boxes for clarity

**Infographics:**
- Use circular progress indicators
- Timeline with connected nodes
- Process flows with directional arrows
- Comparison tables with alternating row colors

### 7. Animation & Transitions

**Slide Transitions:**
- Use "Morph" transition for smooth object movement
- Duration: 0.75 seconds
- Apply to all slides for consistency

**Object Animations:**
- Entrance: Fade + Slight Zoom (0.5s)
- Emphasis: Pulse for key metrics (0.3s)
- Exit: Fade out (0.3s)
- Sequence: Top to bottom, left to right

### 8. Advanced Elements to Add

#### Cover Slide Enhancement
```
Elements to add:
1. Full-bleed background image (abstract tech/business)
2. Dark gradient overlay (60% opacity)

4. Main title (centered, large, bold)

```
Create impactful section breaks:

- Minimal text, maximum impact

#### Data Slides

Enhanced data presentation:
- Chart title with icon
- Color-coded legend
- Callout boxes for key insights
- Source citation in footer
- Comparison indicators (вҶ‘вҶ“ arrows)

## вңЁ Pro Tips

1. **Consistency is Key:** Use the same spacing, colors, and fonts throughout
2. **White Space:** Don't overcrowd slides - let content breathe
3. **Alignment:** Use guides and grids (View вҶ’ Guides)
4. **Contrast:** Ensure text is readable on all backgrounds
5. **Hierarchy:** Make important information stand out
6. **Simplicity:** Complex design doesn't mean cluttered
7. **Brand Alignment:** Ensure SinVedha branding is consistent
